To download our game code from the repository at https://github.com/neo2303241/wheresmymoney

Also download download an IDE or text editor such as Thonny (https://thonny.org) and install the latest version of Python (https://www.python.org).

To test the code extract the code downloaded from GitHub and run the main.py file from your text editor or IDE of choice. Make sure you download PyGame and PyMunk first to ensure you have the correct dependencies.

PyGame - a set of modules for Python allowing the user to write video games.

PyMunk - a 2d physics library for PyGame.

If you would like to deploy changes to the game email us at wheresmymoneyCi103@gmail.com with your changes and we will consider adding you to the Github repository.
